css继承特性主要指文本方面的继承，盒模型相关的属性基本没有继承特性

每个属性在定义的时候给出这个属性是否具有继承性，如果一个属性具有继承性，当他没有设置具体值的时候回从他的父元素继承该属性

如果一个设置的属性不具有继承性， 也可以设置属性值为inherit来继承他的父元素设置的同名属性

## 所有元素可继承
```
visibility
cursor
```

## 终级块元素可继承的
```
text-align
text-indent
```

## 内联元素可继承的
```
font-family/size/style/weight
color
line-height
letter-spacing 字间距
word-spacing 词间距
white-space 对文本中的空白如何处理
```
## 列表元素可继承的
```
list-style
```
