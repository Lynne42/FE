# [omit.js](https://github.com/benjycui/omit.js/)
功能函数， 用于获取一个去掉了指定属性的对象的浅拷贝值
```
var omit = require('omit.js');
omit({ name: 'Benjy', age: 18 }, [ 'name' ]); // => { age: 18 }
```
